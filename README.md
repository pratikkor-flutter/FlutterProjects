# Flutter_Assignment_03

### Objective : <br>
Assignment 03 aims to the understand the concepts of sizedBox, Container, Row & Column.
<br>

### Description : <br>
<p align="justify">
In this, one Row contains three Columns and each of them containing Container Widget & ElevatedButton Widget. Widgets in Row are aligned by SpaceAround property and Widgets within Columns are aligned at Center.
